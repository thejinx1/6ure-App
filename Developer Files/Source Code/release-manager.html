6ure™ App Release Manager

Generate the update manifest that every installed app reads from one stable URL.

Host latest.json at a stable HTTPS URL. A Discord CDN link is fine for the package URL, but Discord signed attachment links can expire; re-uploading latest.json with a fresh package link is required for each release.
Choose the package file to calculate SHA-256.
Put update-config.json into the app before building the setup. Then every installed copy will check the same manifest URL and receive future releases from the values you generate here.
